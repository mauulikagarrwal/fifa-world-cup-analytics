# FIFA World Cup Analytics Project Plan

## Objective

Analyze FIFA World Cup data to discover trends using Python.

## Skills

- Pandas
- NumPy
- Matplotlib
- Seaborn
- Data Cleaning
- Exploratory Data Analysis

## Questions to Answer

- Which team scored the most goals?
- Which player had the most assists?
- Which stadium hosted the highest-scoring matches?
- Which continent performed best?
- How has goal scoring changed over different tournaments?
