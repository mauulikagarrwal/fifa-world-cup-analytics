# FIFA World Cup Analytics

## 📌 Project Overview

This project analyzes FIFA World Cup data using Python to uncover trends in team performance, player statistics, match outcomes, and tournament insights.

## 🛠 Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 🎯 Objectives

- Clean real-world football data
- Perform Exploratory Data Analysis (EDA)
- Create insightful visualizations
- Draw meaningful conclusions

## 📂 Project Structure

data/ – Raw datasets

notebooks/ – Jupyter notebooks

src/ – Python scripts

images/ – Graphs and charts

reports/ – Final reports
